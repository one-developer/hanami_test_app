# Hanami Test Application

Based on [Hanami](https://github.com/hanami/hanami) v0.9.2

## Installation & usage

```bash
$ git clone git@github.com:one-developer/hanami_test_app.git
$ cd hanami_test_app && bundle install
$ hanami db prepare
$ open http://localhost:2300
```
